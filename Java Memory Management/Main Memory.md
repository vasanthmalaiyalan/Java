👉 Computerல **CPU** தான் எல்லா வேலைக்கும் boss 🧠

👉 CPUக்கு:

* RAM access பண்ணுவது ⚡ fast
* HDD/SSD access பண்ணுவது 🐢 slow

👉 அதனால்:
➡️ running programs எல்லாம் **RAMல load ஆகும்**

---

# 🔹 ஏன் app second time fastஆ open ஆகுது?

👉 நீ observe பண்ணிருப்பே 👇

1. First time open → slow 😐
2. Close → மீண்டும் open → fast 😲

👉 ஏன்?

👉 First time:
➡️ App storage (SSD/HDD)ல இருந்து load ஆகும்

👉 Second time:
➡️ RAMல already இருக்கிறது

✔️ அதனால் fast ⚡

👉 இதை:
➡️ **cache / buffer effect** என்று சொல்வாங்க

---

# 🔹 Main Memory overview

👉 Main memoryல முக்கியமானது:

## ✅ RAM (most important)

👉 இது தான்:

* running apps data store பண்ணும்
* fast access தரும்

👉 RAM அதிகமா இருந்தா:
✔️ performance நல்லா இருக்கும்

---

# 🔹 RAM எப்படி performance affect பண்ணுது?

👉 Situation 1:

* RAM free இருக்குது
  ✔️ app smooth run

👉 Situation 2:

* RAM full 😬
  ❌ app slow
  ❌ lag
  ❌ hang

👉 அதனால் தான்:
👉 heavy apps run பண்ணும்போது slow ஆகுது

👉 இது **Activity Monitor (Mac)**

👉 இதில்:

* எந்த app எவ்வளவு memory use பண்ணுது
* total RAM எவ்வளவு use ஆகுது

👉 கீழே important metrics:

* **Physical Memory** → total RAM
* **Memory Used** → currently used
* **Cached Files** → reusable memory
* **Swap Used** → RAM போதாத போது disk use

👉 Swap அதிகமா இருந்தா:
❌ system slow ஆகும்

---

# 🔹 RAM full ஆயிடுச்சுனா என்ன ஆகும்?

👉 OS என்ன பண்ணும்?

➡️ RAM இல்லனா:

* disk (SSD/HDD) use பண்ணும்

👉 இதை:
➡️ **swap memory**

👉 ஆனால்:
❌ இது slow

---

# 🔹 RAM = Volatile memory

👉 முக்கியமான concept ⚠️

* power off → data gone ❌

👉 அதனால்:

* unsaved data loss ஆகும்

---

# 🔹 ROM கூட main memoryல இருக்குது

👉 இது थोड़ा confusing point 😄

### ✅ ROM (Read Only Memory)

* non-volatile
* power off ஆனாலும் data இருக்கும்

👉 இதில் என்ன இருக்கும்?

➡️ computer start ஆக தேவையான instructions (boot code)

---

# 🔹 Important correction (Fun fact 😄)

👉 நாம usually:
➡️ “Main memory = RAM” சொல்வோம்

👉 ஆனால் technically:

👉 Main memory = RAM + ROM

---

# 🔥 Final Deep Understanding

👉 Main memoryல:

* RAM → fast, temporary
* ROM → permanent, startup

👉 CPU:
➡️ RAMயே அதிகமாக use பண்ணும்

---

# 💡 Real-world takeaway

👉 நல்ல performance வேண்டும்னா:

* RAM போதுமான அளவு இருக்கணும்
* unnecessary apps close பண்ணணும்
