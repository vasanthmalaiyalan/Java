👉 இது தான் students அதிகம் confuse ஆகுற இடம் 👇

### ✅ Memory (RAM)

* temporary
* fast ⚡
* power இல்லனா data போயிடும்

### ✅ Storage (HDD / SSD)

* permanent
* slow compared to RAM
* power இல்லாமலும் data இருக்கும்

---

# 🔹 Storage types

👉 இரண்டு main storage:

* HDD (Hard Disk Drive)
  → magnetic storage

* SSD (Solid State Drive)
  → EEPROM (electrical storage)

👉 இவை:

✔️ long-term data save பண்ணும்
✔️ power இல்லாமலும் data இருக்கும்

---

# 🔹 RAM முக்கிய point ⚠️

👉 RAM:

* electricity தேவை
* system off பண்ணினா data clear

👉 அதனால் தான்:

* restart பண்ணினா apps close ஆகுது
