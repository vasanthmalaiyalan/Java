# 🔹 Simple difference

| Concept | Meaning                  |
| ------- | ------------------------ |
| Thread  | execution flow           |
| Stack   | memory for that thread   |
| Frame   | method data inside stack |

---

# 🔹 Easy analogy 🧠

👉 Thread = ஒரு worker 👨‍🔧
👉 Stack = அவன் வேலை மேசை 🧑‍💻
👉 Frame = ஒவ்வொரு task file 📄

👉 worker (thread) வேலை செய்யும்போது:

* table (stack) use பண்ணுவான்
* files (frames) add/remove ஆகும்
