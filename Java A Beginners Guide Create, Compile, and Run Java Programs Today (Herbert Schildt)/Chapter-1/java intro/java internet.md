## 🔹 2. Applets (old concept)

👉 Applet = browser-ல run ஆகும் Java program

👉 Example:

* Calculator
* Form handling

👉 எப்படி work ஆகும்?

* Browser → auto run

## 🔹 3. Security 🔐

👉 Internet biggest problem:

* Virus
* Malware

👉 Java solution:

✔️ Program-ஐ sandbox-ல run பண்ணும்
✔️ System access restrict பண்ணும்

👉 அதனால்:
💡 Safe execution

---

## 🔹 4. Portability 🌍

👉 Internet = different systems

👉 Problem:

* Same app எல்லா system-ல run ஆகணும்

👉 Java solution:
✔️ Bytecode + JVM

👉 Result:
💡 One program → எல்லா இடத்திலும் run ஆகும்
