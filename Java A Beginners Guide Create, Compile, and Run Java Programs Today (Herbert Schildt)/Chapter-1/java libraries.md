# 🧠 PART 1: Java Class Libraries (முக்கிய concept)

---

## 🔹 1. Java Class Library என்றால் என்ன?

👉 **Java-ல already ready-ஆ இருக்கும் classes + methods collection**

👉 Example:

```java
System.out.println("Hello");
```

👉 இங்க:

* `System` → library class
* `println()` → library method

---

## 🎯 Meaning

👉 நீங்க எல்லாம் எழுத வேண்டிய அவசியம் இல்லை

👉 Already Java provide பண்ணும் ✔️

# 🔥 2. என்ன என்ன libraries இருக்கு?

👉 Java library support பண்ணும்:

* Input / Output (I/O)

* GUI

## 🔹 Example packages

## ✅ 10. Inches to meters

👉 formula:

```text
1 meter = 39.37 inches
```

👉 code:

```java
class InchToMeter {
  public static void main(String[] args) {
    int counter = 0;

    for(int inch = 1; inch <= 144; inch++) {
      double meter = inch / 39.37;

      System.out.println(inch + " inches = " + meter + " meters");

      counter++;

      if(counter == 12) {
        System.out.println();
        counter = 0;
      }
    }
  }
}
```
