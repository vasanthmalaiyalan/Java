# 📘 Moving Beyond Applets (Simple Tamil)

## 🕰️ Old Java world

👉 Java ஆரம்ப காலத்தில்:

* Internet புதுசு
* Browser development fast
* Smartphone இல்ல

👉 அந்த நேரத்தில்:
💡 **Applets = Java-வின் hero feature**

---

# 🌐 Applets என்ன ஆனது?

👉 Applet:

* Browser-ல run ஆகும் Java program

👉 Problem:

❌ Browser plugin தேவை
❌ Browsers support குறைய ஆரம்பிச்சது

👉 Result:

* Slowly useless ஆகி போச்சு

---

## ⚠️ Official removal

👉 Java versions:

* JDK 9 → Applet deprecated (use பண்ணாதீங்க warning)
* JDK 11 → Completely removed
* JDK 17 → API கூட remove ஆக போகுது

💡 Meaning:
👉 Applets dead ❌

---

# 🔄 Alternative வந்தது

## 🚀 Java Web Start

👉 இது என்ன?

* Internet-ல இருந்து app download ஆகும்
* Browser இல்லாமல் run ஆகும்

👉 Difference:

| Applet             | Web Start        |
| ------------------ | ---------------- |
| Browser inside run | Separate app run |
| Small apps         | பெரிய apps       |

---

## ⚠️ இதுவும் remove ஆயிடுச்சு

👉 JDK 11:
❌ Web Start support removed

---

# 🤔 அப்போ இப்ப Java apps எப்படி run?

👉 Modern solution 👇

---

## 🔹 1. jlink (JDK 9)

👉 என்ன பண்ணும்?

* Custom runtime create
* உங்கள் app + required JRE சேர்த்து

👉 Benefit:
✔️ Lightweight app

---

## 🔹 2. jpackage (JDK 16)

👉 என்ன பண்ணும்?

* Installable app create

👉 Example:

* .exe (Windows)
* .deb / .rpm (Linux)

👉 Result:
✔️ Normal software மாதிரி install

---

# 🧠 Important point

👉 இந்த deployment tools:

* Advanced topic
* Beginner-க்கு தேவையில்லை

👉 இந்த book-ல்:
✔️ Local-ல run பண்ணுறதுக்கு மட்டும் focus

---

# 🔚 Final Big Picture

👉 Java evolution:

```
Applets ❌ → Web Start ❌ → Modern tools (jlink, jpackage) ✔️
```

👉 அதனால்:
💡 Java still alive & evolving
