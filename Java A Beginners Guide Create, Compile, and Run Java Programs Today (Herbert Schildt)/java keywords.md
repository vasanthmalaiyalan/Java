# 🧠 1. Keyword என்றால் என்ன?

👉 **Keyword = Java-ல் special meaning இருக்கும் reserved word**

👉 Example:

```java
int a;
if(a > 0)
```

👉 இங்க:

* `int`, `if` → keywords

---

# ❗ முக்கிய rule

👉 Keywords:

❌ variable name ஆக use பண்ண முடியாது

---

## ❌ Wrong

```java
int if = 10;   ❌
```

---

## ✔️ Correct

```java
int value = 10; ✔️
```
