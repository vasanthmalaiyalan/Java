## 🧠 The General Form of a Class – Explanation (Tamil)

### 🔹 Class define பண்ணுவது என்ன?

👉 Class உருவாக்கும்போது:

* அதில் என்ன variables இருக்கணும்
* அந்த variables மீது என்ன methods வேலை செய்யணும்
  👉 இதை நீ specify பண்ண வேண்டும்

---

### 🔹 Real-world class எப்படி இருக்கும்?

👉 சில simple class:

* variables மட்டும் இருக்கலாம்
* அல்லது methods மட்டும் இருக்கலாம்

👉 ஆனால் real-worldல:

* variables + methods இரண்டும் இருக்கும்

---

### 🔹 Class எப்படி create பண்ணுவது?

👉 `class` keyword use பண்ணி define பண்ணுவோம்

```java
class classname {
    // variables
    type var1;
    type var2;

    // methods
    type method1(parameters) {
        // code
    }
}
```

---

### 🔹 Class structure என்ன?

👉 ஒரு class உள்ளே இரண்டு முக்கியமானவை:

1. **Instance variables**

   * data store பண்ணும்

2. **Methods**

   * அந்த data மீது வேலை செய்யும் code

---

### 🔹 Important design rule ⚠️

👉 நல்ல class:

* ஒரு logical entity மட்டும் represent பண்ணணும்

📌 Example:

* "Student class" → name, age ✔️
* அதே classல stock market info ❌

👉 காரணம்:

* unrelated data சேர்த்தால் code messy ஆகும்

---

### 🔹 main() method பற்றி

👉 இதுவரை பார்த்த class:

* எல்லாம் `main()` மட்டும் இருந்தது

👉 ஆனால்:

* class general formல `main()` இருக்க வேண்டிய அவசியம் இல்லை

👉 `main()` தேவை:

* program start ஆகும் classக்கு மட்டும்

👉 சில Java applications:

* `main()` இல்லாமலும் இருக்கும்
