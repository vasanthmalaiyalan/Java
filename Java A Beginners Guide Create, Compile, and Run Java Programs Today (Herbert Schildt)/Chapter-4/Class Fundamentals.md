## 🧠 Class Fundamentals – Explanation (Tamil)

### 🔹 Javaல எல்லாமே class உள்ளே தான் நடக்கும்

👉 Java programல வரும் எல்லா code-மும் **class உள்ளே தான் இருக்கும்**

---

### 🔹 இதுவரை simple class தான் பார்த்தோம்

👉 இப்போ வரை:

* மிகவும் basic/simple class மட்டும் use பண்ணியிருக்கோம்
  👉 இன்னும் நிறைய features இருக்கிறது (class மிகவும் powerful)

---

### 🔹 Class என்றால் என்ன?

👉 Class = ஒரு **template / plan**

👉 அது define பண்ணும்:

* என்ன data இருக்கணும்
* அந்த data மீது என்ன code (methods) வேலை செய்யணும்

---

### 🔹 Object எப்படி உருவாகிறது?

👉 Java:

* class-ஐ use பண்ணி object உருவாக்கும்

👉 Object = class-ன் instance

📌 அதாவது:

> Class = plan
> Object = அந்த plan அடிப்படையில் உருவாக்கப்பட்ட ஒன்று

---

### 🔹 முக்கியமான point ⚠️

👉 Class = **logical abstraction**

* இது idea மாதிரி (memoryல real இல்லை)

👉 Object உருவான பிறகு தான்:

* அது memoryல real ஆக இருக்கும்

📌 அதாவது:

> Class மட்டும் இருந்தா → memoryல எதுவும் இல்லை
> Object create பண்ணினால் → memoryல உருவாகும்

---

### 🔹 Class members என்ன?

👉 Class உள்ளே இருக்கும்:

* variables
* methods

👉 இவை எல்லாம் சேர்த்து:

> “members of the class”

---

### 🔹 Data members (Instance variables)

👉 Class உள்ளே இருக்கும் variables:

* instance variables என்றும் சொல்லுவாங்க

---

## ⚡ One-line Summary

👉 Class = plan
👉 Object = real instance
👉 Variables + Methods = members
👉 Variables = instance variables
