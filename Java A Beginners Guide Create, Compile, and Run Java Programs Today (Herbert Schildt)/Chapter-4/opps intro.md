## 📘 Paragraph Explanation (Tamil)

👉 Java கற்றுக்கொள்ள தொடர்ந்து போகணும்னா
**class பற்றி கண்டிப்பா தெரிஞ்சிருக்கணும்**

---

👉 Class தான் Javaவின் **முக்கியமான அடிப்படை (essence)**

👉 முழு Java language:

* class அடிப்படையில்தான் உருவாக்கப்பட்டுள்ளது

👉 ஏன் என்றால்:

* class தான் object எப்படி இருக்கணும் என்று define பண்ணும்

---

👉 அதனால்:

* class = object-oriented programming (OOP)க்கு அடிப்படை

---

👉 ஒரு class உள்ளே:

* data இருக்கும்
* அந்த data மீது வேலை செய்யும் code இருக்கும்

👉 அந்த code:

* methods உள்ளே இருக்கும்

---

👉 Class + Object + Method:

* இந்த மூன்றும் Javaல மிக முக்கியமானவை

👉 அதனால் தான்:

* இந்த chapterல introduce பண்ணப்பட்டுள்ளது

---

👉 இந்த concepts basicஆ புரிந்தால்:

* இன்னும் advanced programs எழுத முடியும்
* அடுத்த chapterல வரும் Java concepts easyஆ புரியும்
