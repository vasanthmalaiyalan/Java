👉 **Primitive = simple binary values (object அல்ல)**

👉 Meaning:

✔️ direct value store பண்ணும்

# 🎯 3. Java-வில் 8 Primitive Types

---

## 📊 Full list

| Type    | Meaning                    |
| ------- | -------------------------- |
| boolean | true / false               |
| byte    | small integer              |
| short   | medium integer             |
| int     | normal integer             |
| long    | பெரிய integer              |
| float   | decimal (single precision) |
| double  | decimal (high precision)   |
| char    | character                  |

---

# 🔍 4. ஒவ்வொன்றும் simple explain

---

## 🔹 boolean

```java
boolean flag = true;
```

👉 only:
✔️ true / false

---

## 🔹 byte

```java
byte b = 100;
```

👉 small integer
👉 memory save

---

## 🔹 short

```java
short s = 1000;
```

👉 medium size

---

## 🔹 int (most used 🔥)

```java
int x = 10;
```

👉 default integer

---

## 🔹 long

```java
long big = 10000000000L;
```

👉 பெரிய values

---

## 🔹 float

```java
float f = 10.5f;
```

👉 decimal (less precision)

---

## 🔹 double (most used 🔥)

```java
double d = 10.5;
```

👉 high precision decimal

---

## 🔹 char

```java
char ch = 'A';
```

👉 single character

# 🧠 5. Why primitive types not objects?

👉 காரணம்:

✔️ performance
✔️ speed
✔️ memory efficiency

# 🔥 6. Important concept (very important 🔥)

👉 Java:

✔️ fixed size define பண்ணி இருக்கு

👉 Example:

👉 `int`:

✔️ எல்லா system-ல same size

---

# 🎯 Why important?

👉 portability

👉 Java promise:

```text
Write Once → Run Anywhere
```

👉 அதனால்:

✔️ type behavior same everywhere

---

# 🧠 Example

👉 Windows:

```java
int x = 10;
```

👉 Linux:

```java
int x = 10;
```

👉 Same behavior ✔️

# ⚠️ Trade-off

👉 strict size:

✔️ portability
❌ slight performance cost
