சூப்பர் 🔥 இப்போ நீங்க **Literals (constants)** கிட்ட வந்துட்டீங்க — இது basic ஆனாலும் மிகவும் முக்கியம் 💯
நான் இதை **simple Tamil + examples + rules + tricks**ல explain பண்ணுறேன் 👇

---

# 🧠 1. Literal என்றால் என்ன?

👉 **Literal = fixed value (direct-ஆ எழுதுற value)**

👉 Example:

```java
int x = 100;
```

👉 இங்க:

👉 `100` → literal ✔️

---

# 🔥 2. Literal = Constant

👉 change ஆகாத value

👉 program-ல direct use பண்ணுறோம்

---

# 🎯 3. Types of literals

---

## 🔹 1. Integer literals

```java
10
-100
```

👉 decimal இல்லாத number

---

## 🔹 2. Floating-point literals

```java
11.123
3.14
```

👉 decimal values

---

## 🔹 3. Character literals

```java
'A'
'%'
```

👉 single quotes ⚠️

---

## 🔹 4. Boolean literals

```java
true
false
```

---

# 🔥 4. Default types (மிக முக்கியம் 🔥)

---

## 🔹 Integer → default = int

```java
10   → int
```

👉 long வேண்டுமா?

```java
10L  ✔️
```

---

## 🔹 Floating → default = double

```java
10.5 → double
```

👉 float வேண்டுமா?

```java
10.5F ✔️
```

---

# ⚠️ Important examples

---

## ❌ Wrong

```java
float f = 10.5;   ❌
```

---

## ✔️ Correct

```java
float f = 10.5F; ✔️
```

---

## ❌ Wrong

```java
long x = 10000000000; ❌
```

---

## ✔️ Correct

```java
long x = 10000000000L; ✔️
```

---

# 🔥 5. Scientific notation

👉 big/small numbers:

```java
double x = 1.2e3;   // 1200
double y = 1.2e-3;  // 0.0012
```

---

# 🔥 6. Underscore feature (cool 🔥)

👉 readability improve பண்ண:

```java
int num = 123_45_1234;
```

👉 same as:

```text
123451234
```

---

## 🎯 Use case

👉 IDs, numbers, codes

---

# 🧠 7. Type compatibility

👉 integer literal assign பண்ணலாம்:

✔️ byte
✔️ short
✔️ char

👉 if range உள்ளே இருந்தா ✔️

---

## ✔️ Example

```java
byte b = 100;   ✔️
```

---

## ❌ Example

```java
byte b = 200;   ❌ (range exceed)
```
