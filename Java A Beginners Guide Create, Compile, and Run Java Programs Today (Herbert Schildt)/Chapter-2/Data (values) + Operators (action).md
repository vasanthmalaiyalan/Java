# 🧠 Big idea (முதலில் புரிஞ்சிக்கோ)

👉 Programming =

```text
Data (values) + Operators (actions)
```

👉 Example:

```java
int a = 10;
int b = 5;
int c = a + b;
```

👉 Data → 10, 5
👉 Operator → `+`

---

---

# 🎯 1. Primitive Types (முக்கியம் 🔥)

👉 Java basic data types:

* `int` → integer
* `double` → decimal
* `char` → character
* `boolean` → true/false

👉 இவை:

👉 **building blocks of Java**

---

# 🎯 2. Literals

👉 literal = direct value

```java
10      // int literal
10.5    // double literal
'a'     // char literal
true    // boolean literal
```

---

# 🎯 3. Variable Initialization

👉 variable declare + value assign

```java
int x = 10;
```

👉 இது:

✔️ declare + initialize

---

# 🎯 4. Scope (மிக முக்கியம் 🔥)

👉 variable எங்க use பண்ணலாம்?

👉 Example:

```java
if(true) {
    int x = 10;
}
```

👉 outside use பண்ண முடியாது ❌

👉 இது தான் scope

---

# 🎯 5. Arithmetic Operators

👉 math operations:

* `+` → add
* `-` → subtract
* `*` → multiply
* `/` → divide
* `%` → remainder

---

# 🎯 6. Relational Operators

👉 compare பண்ண:

* `<`, `>`, `<=`, `>=`
* `==`, `!=`

👉 output:

✔️ true / false

---

# 🎯 7. Logical Operators

👉 conditions combine பண்ண:

* `&&` → AND
* `||` → OR
* `!` → NOT

---

# 🎯 8. Assignment Operators

👉 value assign:

```java
x = 10;
```

---

# 🎯 9. Shorthand Assignments

👉 short form:

```java
x += 5;   // x = x + 5
x -= 2;
x *= 3;
x /= 2;
```

---

# 🎯 10. Type Conversion

👉 data type change

---

## 🔹 Automatic

```java
int a = 10;
double b = a;  // auto convert
```

---

## 🔹 Manual (casting)

```java
double x = 10.5;
int y = (int) x;  // 10
```

---

# 🎯 11. Expressions

👉 combination:

```java
int result = (a + b) * 2;
```

👉 expression = calculation
